# Estilo de Correos de Paula

Esta guía resume el tono y la estructura preferida para redactar correos en nombre de Paula.

## Tono general

- Profesional, amable y claro.
- Cercano, sin sonar demasiado informal.
- Directo, pero siempre cordial.
- Sin frases rígidas, excesivamente corporativas o demasiado largas.
- Evitar sonar insistente, agresiva o demasiado técnica.
- El correo debe sentirse natural, como seguimiento humano y no como formato genérico.

## Estructura habitual

1. Saludo:

```text
Estimada [Nombre]:
```

2. Primera línea:

```text
Espero que estés muy bien.
```

3. Desarrollo en párrafos:

El cuerpo debe ir en párrafos completos, no en listas, salvo que Paula pida expresamente una lista o sea indispensable para claridad.

4. Cierre:

```text
Muchas gracias y saludos,
Paula
```

## Plantilla base

```text
Estimada [Nombre]:

Espero que estés muy bien.

[Motivo del correo, explicado de forma clara y amable.]

[Seguimiento, solicitud concreta, contexto o siguiente paso.]

Muchas gracias y saludos,
Paula
```

## Ejemplo de seguimiento

```text
Estimada Toni:

Espero que estés muy bien.

Te escribo para dar seguimiento a la revisión de LexContratos. Quería preguntarte si ya han tenido oportunidad de probar la plataforma y, en su caso, si me pudieras compartir sus comentarios: qué les ha funcionado bien, qué necesita ajustes, en qué puntos han tenido dudas o si han detectado algún error durante el uso.

También me ayudaría mucho saber en qué estatus van con las pruebas y si hay algo puntual que consideren prioritario ajustar antes de avanzar a la siguiente etapa.

Si te parece, podríamos agendar una junta la próxima semana para revisar sus observaciones, resolver dudas y definir los siguientes pasos.

Muchas gracias y saludos,
Paula
```

## Firma sugerida para correo

Opción sobria:

```text
Paula Tarragó
LexContratos
```

Opción profesional:

```text
Paula Tarragó
LexContratos
Asistente contractual para equipos legales
paula@lexcontratos.com
```

Opción más institucional:

```text
Paula Tarragó
LexContratos
Contratos más ágiles, ordenados y listos para revisar
paula@lexcontratos.com
lexcontratos.com
```

## Preferencia actual

Para correos cotidianos, usar el cierre simple:

```text
Muchas gracias y saludos,
Paula
```

La firma institucional puede agregarse desde el cliente de correo, debajo del cierre, cuando convenga reforzar marca o formalidad.
