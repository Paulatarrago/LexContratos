# Checklist de validación para Tony

## Acceso

- [ ] Pude entrar con mi usuario.
- [ ] Entendí dónde está la app privada.
- [ ] Entendí que mi información está separada por usuario.

## Mis Documentos

- [ ] Entendí la diferencia entre carpetas normales y carpetas del sistema.
- [ ] Pude abrir `Mis Documentos`.
- [ ] Pude abrir `Formatos del sistema`.
- [ ] Pude abrir `Catálogos del sistema`.
- [ ] Pude crear una carpeta.
- [ ] Pude crear una subcarpeta.
- [ ] Pude renombrar una carpeta.
- [ ] Pude seleccionar varios documentos con Shift/Cmd/Ctrl.
- [ ] Pude buscar un documento.

## Formatos del sistema

- [ ] Entendí que los formatos del sistema son bases.
- [ ] Entendí que se trabaja sobre copias, no sobre el formato base.
- [ ] Como administradora, pude renombrar o editar un formato.
- [ ] La advertencia antes de modificar un formato del sistema fue clara.

## Flujo de contrato

- [ ] Entendí que primero debo detectar campos editables.
- [ ] Entendí dónde se cargan los documentos de cada parte.
- [ ] Entendí cuándo usar `Integrar faltantes`.
- [ ] Entendí para qué sirve la revisión crítica IA.
- [ ] Entendí cuándo exportar Word.
- [ ] Entendí cuándo exportar PDF para firma.

## Documentos por parte

- [ ] Pude cargar documentos del cliente/contratante.
- [ ] Pude cargar documentos de la contraparte/proveedor.
- [ ] Entendí que los documentos no se mezclan entre partes.
- [ ] La extracción de datos fue clara.
- [ ] Los datos faltantes quedaron visibles.

## Membretes

- [ ] Entendí dónde están los membretes.
- [ ] Entendí cómo seleccionar un membrete.
- [ ] Entendí que el membrete se compone de logo/imagen y pie de página.
- [ ] El resultado visual en Word/PDF me parece usable.

## Exportación

- [ ] El Word salió con formato usable.
- [ ] El PDF para firma se entiende.
- [ ] El documento queda listo para enviarse a firma externa.

## Observaciones

Anotar:

- Lo que no entendí.
- Dónde me atoré.
- Qué botón cambiaría de nombre.
- Qué flujo se siente largo.
- Qué pantalla se siente saturada.
- Qué dato faltante debería reconocer mejor.
- Qué formato falta subir.
- Qué membrete falta cargar.
