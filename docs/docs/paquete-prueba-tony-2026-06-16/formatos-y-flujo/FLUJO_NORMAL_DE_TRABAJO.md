# Flujo normal de trabajo en LexContratos

Este flujo es el que conviene mostrar en la junta. Está pensado para que Tony vea la lógica completa de la app sin meterse todavía a administración avanzada.

## 1. Entrar a la app

1. Abrir `lexcontratos.com`.
2. Iniciar sesión con una cuenta activa.
3. Entrar al área privada de trabajo.

## 2. Seleccionar un formato

1. Dar clic en `Seleccionar formato`.
2. Buscar el tipo de contrato.
3. Elegir un formato.
4. Confirmar que se abre como documento editable.

Nota: los formatos del sistema deben funcionar como base. El usuario trabaja sobre una copia, no sobre el formato base, salvo que sea administradora y confirme que desea modificar un archivo del sistema.

## 3. Guardar como

1. Dar clic en `Guardar como`.
2. Elegir ubicación.
3. Crear carpeta si hace falta.
4. Renombrar el documento.
5. Guardar.

Ubicaciones principales:

- `Mis Documentos`
- `Clientes`
- `Proveedores`
- `Empresas del Grupo`
- `Documentos de las partes`
- `Otros`
- `Formatos del sistema` para administradoras
- `Catálogos del sistema` para administradoras

## 4. Aplicar membrete

1. Abrir el catálogo de membretes.
2. Seleccionar el membrete.
3. Revisar que se vea correctamente en el documento.
4. Si no debe llevar membrete, usar la opción de quitarlo.

## 5. Detectar campos editables

1. Dar clic en `Detectar campos editables`.
2. Esperar a que la app identifique campos rellenables.
3. Verificar que el sistema detectó datos de partes, fechas, montos, representantes, domicilios y otros campos relevantes.

Este paso es obligatorio antes de cargar e integrar datos, porque la app necesita saber qué espacios puede llenar.

## 6. Datos y documentos

1. Dar clic en `Datos y documentos`.
2. Cargar documentos en la caja de la parte correspondiente.
3. Usar documentos ficticios del paquete para la prueba.
4. Dar clic en `Extraer e integrar datos`.

La lógica deseada es que, según el formato seleccionado, las cajas muestren nombres de partes adecuados al contrato, no nombres genéricos que confundan.

## 7. Integrar faltantes

1. Revisar los campos que no fueron encontrados en documentos.
2. Llenar varios campos pendientes.
3. Usar `Integrar faltantes`.
4. Verificar que los campos ya integrados desaparezcan de la lista de pendientes.

## 8. Revisión crítica IA

1. Ejecutar `Revisión crítica IA`.
2. Revisar observaciones.
3. Aceptar o descartar cambios individualmente.
4. No aplicar cambios en bloque si hay observaciones que no proceden.

## 9. Exportar

1. Exportar Word para revisión editable.
2. Exportar PDF para firma.
3. Enviar el PDF o Word a la herramienta externa de firma que el equipo decida usar.

## 10. Manejo de documentos en Mis Documentos

En el explorador:

- Clic normal: selecciona un documento.
- Shift + clic: selecciona un rango.
- Cmd/Ctrl + clic: agrega o quita elementos de la selección.
- Doble clic: abre carpeta o documento.
- Clic derecho: abre acciones disponibles.

Acciones esperadas:

- Crear carpeta.
- Renombrar.
- Mover.
- Copiar.
- Eliminar con confirmación.
- Buscar.
- Cambiar vista.
