## Tipo de tema

Marcar uno:

- [ ] No encontré el botón o función.
- [ ] El texto no se entiende.
- [ ] El diseño se ve raro.
- [ ] El flujo tiene pasos de más.
- [ ] No guardó.
- [ ] No extrajo datos.
- [ ] No integró datos.
- [ ] El Word/PDF salió mal.
- [ ] Error técnico.
- [ ] Sugerencia de mejora.

## Prioridad

- [ ] Alta: bloquea el uso.
- [ ] Media: se puede usar, pero confunde.
- [ ] Baja: mejora deseable.
