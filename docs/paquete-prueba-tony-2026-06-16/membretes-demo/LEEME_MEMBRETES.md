# Membretes demo

Estos archivos son ficticios y sirven para enseñar el flujo de membretes sin usar logos reales.

## Formato recomendado

Para pruebas visuales:

- SVG si se quiere un logo limpio y escalable.
- PNG si ya tienen imagen institucional lista.
- JPG solo si el archivo viene de diseño o escaneo y no hay versión mejor.

Para producción:

- Preferir SVG o PNG transparente.
- Usar archivos con buena resolución.
- Nombrar cada membrete con empresa y versión, por ejemplo:
  - `Grupo_Cocei_Membrete_2026.png`
  - `Empresa_Operativa_Norte_Membrete_2026.svg`

## Qué mostrar en la junta

1. Seleccionar un membrete del catálogo.
2. Aplicarlo al contrato.
3. Quitar el membrete.
4. Explicar qué una administradora podrá cargar o actualizar membretes autorizados.
