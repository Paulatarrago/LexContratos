Documento inicial

Aviso de privacidad

LexContratos se encuentra en uso privado inicial. Este aviso describe, de forma preliminar, cómo se pretende tratar la información cargada para preparar contratos y expedientes.

Responsable

Pendiente de completar con la razón social, domicilio y datos de contacto del responsable del tratamiento antes de uso comercial o apertura pública.

Datos tratados

Podrán cargarse datos de identificación, fiscales, corporativos, notariales, de representación legal, contacto y documentos soporte necesarios para preparar contratos.

Finalidades

Preparar contratos, organizar expedientes, extraer datos para revisión, conservar versiones editables y habilitar flujos de exportación o firma electrónica cuando estén configurados.

Control del usuario

El usuario autorizado conserva control sobre la información, revisión de datos, contenido contractual y versión final. No se debe cargar información innecesaria para el contrato o expediente.

Documento inicial

Términos de uso

Estos términos son una base inicial para uso privado interno. Deben revisarse y completarse antes de ofrecer LexContratos comercialmente o abrirlo al público.

Uso permitido

LexContratos se ofrece como herramienta de apoyo operativo para preparar contratos, organizar expedientes y reducir captura repetitiva. No sustituye la revisión profesional ni la decisión final del usuario.

Acceso privado

El acceso está limitado a usuarios autorizados. Las cuentas pueden requerir activación manual, licencia vigente o aprobación administrativa.

Información cargada

El usuario es responsable de cargar únicamente documentos necesarios, revisar los datos extraídos y validar la versión final antes de exportar o enviar a firma.

Integraciones

Los flujos de firma electrónica, correo y extracción documental dependen de configuraciones externas. Las funciones no configuradas deben considerarse en preparación.

Manual de uso

Guía rápida para trabajar contratos

LexContratos es un asistente eficaz para extraer datos validables, prellenar generales y reducir captura repetitiva. Tú conservas el control de datos, clausulado, expediente y versión final.

1
Elige tu punto de partida

Selecciona un machote precargado o importa tu propio contrato como plantilla.

2
Trabaja siempre sobre una copia

Duplica la plantilla para editar sin alterar la versión base.

3
Organiza el expediente

Elige dónde guardar el contrato por cliente, proveedor, asunto o carpeta interna.

4
Carga documentos de cada parte

Arrastra actas, poderes, constancias fiscales, identificaciones oficiales, correos, Word, PDF, Excel o imágenes en la caja correspondiente.

5
Extrae, revisa y corrige

Usa “Extraer y revisar datos” para llenar campos; valida la información y completa manualmente lo que falte.

6
Finaliza con criterio profesional

Edita el clausulado, revisa campos pendientes y exporta a Word o prepara el envío a firma.

Consejo práctico

Sube documentos corporativos, constancias fiscales e identificaciones en la caja de cada parte para evitar confusiones entre roles.

Antes de exportar

Revisa la lista de datos pendientes. Si falta información, puedes subir otro documento o capturarla manualmente.

Plantillas base

Las plantillas se protegen. Para modificarlas, trabaja una copia y reemplaza la plantilla base solo cuando estés segura.
